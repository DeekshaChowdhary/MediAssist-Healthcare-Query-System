📡 REST API Reference

POST/api/v1/chat

Send a message and get an AI-generated health response
// Request
{
  "message": "What are symptoms of diabetes?",
  "session_id": "abc123"   // optional, for multi-turn chat
}

// Response
{
  "status": "success",
  "data": {
    "session_id": "abc123",
    "question": "What are symptoms of diabetes?",
    "answer": "Diabetes symptoms include frequent urination...",
    "sources_used": 3,
    "confidence": "87%",
    "latency_ms": 1240.5,
    "timestamp": "14:32:01"
  }
}

POST/api/v1/session/new

Start a new conversation session (returns session_id)
// Response: { "status": "success", "session_id": "uuid-here" }

GET/api/v1/session/<id>/history

Get full conversation history for a session

POST/api/v1/search

Direct semantic vector search — returns matching knowledge base entries without LLM generation
// Request: { "query": "fever treatment", "top_k": 3 }
// Response: { "status": "success", "results": [...] }

DELETE/api/v1/session/<id>/clear

Clear session history

GET/api/v1/health

System health check — shows loaded models and stats
